# Vortex_vision-8-birds_eye_view

## Task Description
This project demonstrates how to transform a perspective view of a business card into a top-down "Bird's Eye View" using OpenCV. The user manually selects the four corners of the card using a mouse callback function, and the program applies a perspective transformation to flatten the image.
